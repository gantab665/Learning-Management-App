# Learning Management Application (LMS)

A simple and modern Learning Management System built using Next.js, Redux Toolkit, AWS, and Clerk Authentication. This application provides an easy learning experience for students, instructors, and admins with role-based access and a clean, responsive interface.

---

## Features

### Core LMS Features

- Course creation and management (modules, lessons, quizzes)
- Upload and stream course videos
- Assignments, submissions, and grading
- Discussion forums for each course
- Student progress tracking

### User Roles

- Student dashboard
- Instructor dashboard
- Admin dashboard

### Authentication and Authorization

- Secure login with Clerk
- Role-based access control (RBAC)
- Session management with JWT tokens

### Performance

- Fast load times with Next.js optimization
- Scalable AWS backend
- Responsive UI across devices

---

## Tech Stack

- Next.js
- React + Redux Toolkit
- TypeScript
- Tailwind CSS
- AWS (S3, Lambda, API Gateway)
- Clerk Authentication
